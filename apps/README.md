# reactForums Apps

Here is containted the three ready-made apps: browser, mobile, and desktop.

## Browser

The browser app is a Tanstack Start app with an exposed REST API to be consumed by all three app instances.

## Mobile

The mobile app is an Expo app that consumes data from the REST API of a browser instance of reactForums.

## Desktop

The native desktop app is an Electron wrapper around the browser app.
